# Privacy Policy

*Last updated: 2026-07-20*

## Summary

archive-forensic is a local Rust library. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

archive-forensic reads only the archive bytes you pass to it. All decompression and analysis happen in memory (with an optional temp-file spill for large inner streams) on your local machine. Nothing is uploaded anywhere.

## Telemetry

archive-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

archive-forensic is open source. You can audit every line of code at [github.com/SecurityRonin/archive-forensic](https://github.com/SecurityRonin/archive-forensic).
